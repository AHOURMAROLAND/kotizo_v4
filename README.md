# Kotizo v4

Application mobile de gestion de cotisations collectives — marché Togo.

- Mobile : React Native + Expo SDK 55
- Backend : Django 5 + DRF + Celery + Redis
- Admin : React.js + Tailwind CSS
- Paiements : PayDunya (Mixx by Yas, Moov Money, T-Money Togo)
- Bot WhatsApp : Evolution API

## Structure

kotizo-backend/   Django 5 + DRF
kotizo-mobile/    React Native Expo
kotizo-admin/     React.js admin web
docker/           Docker Compose configs
docs/             DIM et documentation
